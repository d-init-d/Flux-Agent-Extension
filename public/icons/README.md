- https://cloudconvert.com/svg-to-png
- Or use a tool like Inkscape/GIMP

Required sizes:
- icon-16.png (16x16)
- icon-32.png (32x32)
- icon-48.png (48x48)
- icon-128.png (128x128)

Color scheme:
- Primary: #3B82F6 (blue)
