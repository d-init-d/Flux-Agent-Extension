What is Flux Agent?

A Chrome extension that lets you control your browser with natural language. Record workflows, replay actions, and automate repetitive tasks — all from a clean side panel interface.

Privacy First

Your data stays on your device. Flux Agent only sends data to AI providers you explicitly configure. No tracking, no ads, no data selling.

Open Source

The full source code is available on GitHub. Review every line of code that runs in your browser.
